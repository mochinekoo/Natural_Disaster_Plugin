# 概要
* このプラグインは自然災害情報を取得するプラグインです。

# API
| APIの種類     | 取得元      | URL                      | 備考             |
|------------|----------|--------------------------|----------------|
| 地震情報       | P2P地震情報様 | https://www.p2pquake.net |                |
| 緊急地震速報（予報） | ✖        | ✖                        | 取得可能なAPIが無いため。 |
| 緊急地震速報（警報） | P2P地震情報様 | https://www.p2pquake.net |                |
| 津波の到達予想    | P2P地震情報様 | https://www.p2pquake.net |                |
| 津波の観測情報    | ✖        | ✖                        |                |



# 注意事項
* このプラグインで取得できる情報は、気象庁等から配信されたものを、民間の事業者が再配信し、それを取得しているものです。そのため、タイムラグがあります。
